# Capstone Project

Project build with React.js & Next.js

Live version of [website](http://mloart.amz-x.com)

## Requirements

- NodeJS 12.x LTS
- NPM

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production


### Build 

```bash
npm run build
```

### Test

```bash
npm run serve
```
